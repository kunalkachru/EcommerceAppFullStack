# Contributing

**Last updated:** 2026-07-06

Thank you for interest in ShopEase. This is a **portfolio / demo project** built with agent-assisted development.

## Quick start

1. Read [README.md](README.md) and [docs/SETUP.md](docs/SETUP.md)
2. Try the [live Appetize demo](https://appetize.io/app/b_syzdh2dfef37uy3fyeib33aky4) before cloning
3. Run `npm test` and `npm run verify:secrets-policy` before opening a PR

## Scope

- Bug fixes and doc improvements welcome
- No macOS GitHub Actions on every push (hobby cost constraint)
- Do not commit secrets or `.env` files

## Agent-assisted workflow

See [docs/AGENTIC_DEVELOPMENT.md](docs/AGENTIC_DEVELOPMENT.md) and [AGENTS.md](AGENTS.md) for Cursor subagents used in this repo.
